# Test Data Management

A tool to generate realistic test data for your projects.


## Prerequisite: 

Install Node.js from Company Portal


## Development

To run the project locally, follow these steps:

1.  **Open command prompt and navigate to the project folder within the terminal**:


2.  **Install dependencies (while in tdm/ folder)**:
    ```bash
    npm install
    ```

3.  **Install dependencies (while in tdm/server folder)**:
    ```bash
    npm install
    ```

4.  **Start the frontend server (while in tdm/ folder)**:
    ```bash
    npm run dev
    ```

5.  **Start the development server (while in tdm/server folder)**:
    ```bash
    npm run dev
    ```

6.  Open your browser and navigate to `http://localhost:8080` (or the address shown in your terminal).


## Potential Errors: 

You may run into a Bit9 security blocker when running 'npm run dev'. You will need to submit an access request to allow you to run the tool.
